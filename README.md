##　2.代码库使用说明
### 2.1.首先fork 项目
### 2.2.下载
```
git clone https://github.com/你的用户名/PowerAI
git remote add upstream https://github.com/wangxiaoleiAI/PowerAI
```
### 2.3.从代码库中下拉最新代码
```
git fetch upstream
git merge upstream
```
### 2.4.push到自己的代码仓库中
git push
### 2.5.代码提交至总代码库总github网页上进行requests
